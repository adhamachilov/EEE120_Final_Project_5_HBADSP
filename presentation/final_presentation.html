04 · Logic Design

Boolean Expressions & Gate Architecture

✅ ADMIT Output
ADMIT = B · D · (E + P)
Bed available AND Doctor approved
AND (Emergency OR Payment cleared)
⏳ WAITLIST Output
W = (B·D·E'·P') + (B·D'·E)
Bed+Doctor but no payment/emergency
OR Bed+Emergency but no doctor yet
❌ REJECT Output
REJECT = ADMIT' · WAIT'
All remaining cases — no bed available
or insufficient approvals
Gates Used (≥9)
AND ×4 OR ×3 NOT ×2
4 inputs · 3 outputs
All combinational, no memory

07 · Python Software Prototype

def admission_decision(bed, emergency, doctor, payment): """Evaluates admission using combinational logic.""" admit = bed and doctor and (emergency or payment) waitlist = ( (bed and doctor and not emergency and not payment) or (bed and not doctor and emergency)) reject = not admit and not waitlist return admit, waitlist, reject def run_truth_table(): for i in range(16): b=bool((i>>3)&1); e=bool((i>>2)&1) d=bool((i>>1)&1); p=bool(i&1) admit, wait, rej = admission_decision(b,e,d,p) print(b,e,d,p, admit, wait, rej) def main(): # MAIN MENU "1.
